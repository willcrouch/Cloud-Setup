Set up Google Cloud Instnace
-- Ubuntu 18.10 used in this example
	1vCPU
	Allow HTTP
	Allow HTTPS
	Everything else - default

After initalization, go to SSH dropdown and get *gcloud command line* 

Using GoogleCloudSDK:
Paste command line and hit enter

Putty should launch

Select that you do want to add the key

#Setup Code:

$ sudo apt-get update

$ sudo apt-get upgrade

$ sudo apt-get install r-base

$ sudo su - \-c "R -e \"install.packages('shiny', repos='https://cran.rstudio.com/')\""
--See code list for others, depends on what you are running

$ sudo apt-get install gdebi-core
$ wget https://download3.rstudio.org/ubuntu-14.04/x86_64/shiny-server-1.5.9.923-amd64.deb
$ sudo gdebi shiny-server-1.5.9.923-amd64.deb

At this point should have notice that server is running
